=== WooCommerce Checkout Add-Ons ===
Author: skyverge
Tags: woocommerce
Requires at least: 5.2
Tested up to: 5.6.2
Requires PHP: 7.0

Easily add add-ons to your WooCommerce orders and display them in the Orders admin, the My Orders section, and even order emails!

See https://docs.woocommerce.com/document/woocommerce-checkout-add-ons/ for full documentation.

== Installation ==

1. Upload the entire 'woocommerce-checkout-add-ons' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
